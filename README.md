## Introduction
This project demonstrates the end-to-end deployment of **osTicket**, a widely used open-source ticketing platform, within a cloud-hosted environment. The objective was to build a functional **Service Management** hub from the ground up, starting with web server (IIS) and database (MySQL) configuration through to application lifecycle management. This project simulates the technical backend responsibilities of an IT professional—ensuring that the support infrastructure is secure, stable, and ready to handle user requests, agent workflows, and SLA tracking.

---

## Technical Skills & Tools
* **Ticketing System Management:** osTicket v1.15.8 Deployment, Configuration, and Testing.
* **System Administration:** Windows 11 Pro, User Access Control (ACLs), and Service Management.
* **Web Server Stack:** Internet Information Services (IIS), PHP 7.3.8 Integration, and Extension Management.
* **Database Administration:** MySQL 5.5.62 & HeidiSQL (Schema Setup and User Management).
* **Cloud Infrastructure:** Microsoft Azure Virtual Machine Provisioning (4 vCPUs).
* **Security & Access Control:** Managing NTFS file permissions and security inheritance for application data.
